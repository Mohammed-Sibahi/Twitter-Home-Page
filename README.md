## Twitter Home Page
This project is achieved using HTML5, CSS3, Font-awesome and BOOTSTRAP4.
There are various features implemented here such as

 - Flex Box
 - Buttons
 - Cards
 - Headers
 - Search
 - Font-awesome icons
 - Input
 - footer
